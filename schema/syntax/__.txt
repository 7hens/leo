Leo使用符号来替换所有的关键字
	if 		?
	unless 	?:
	when 	?>
	case 	:>
	else 	::

	while 	%
	until 	%:
	for 	%>
	in 		**
	break 	--
	cont 	++

	func 	$$
	def 	$>
	return 	//

	at 		@
	type 	$
	to 		..
	less_to ...

Leo中不同于JS的符号
	&&	||	<<	<=	==	!=	>=	>>	~	+	-	*	/	\
	&&=	||=	<<=	<==	===	!==	>==	>>=	~=	+=	-=	*=	/=	\=
